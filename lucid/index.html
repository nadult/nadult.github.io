Lucid: compute-shader based software rasterizer for transparency

Lucid is a fully software-based rasterizer which runs on Vulkan's compute shaders. It was designed for efficient rendering of transparent objects. No pre-sorting of geometry is required: all sorting happens in the pipeline.

Lucid rasterizer is quite efficient: compared to hardware-based simple alpha-blended rasterization it is on average only about 3 times slower. It means that it is on par with state-of-the-art software rasterizers even though it has additional sorting phase.

Running: download and unpack lucid_beta.zip and either lucid_scenes.zip or lucid_scenes.7z. In case of problems with missing dlls, make sure that MSVC++ redist is installed. Lucid should run on recent AMD & NVidia GPUs (both integrated & discrete GPUs). At least 2GB of VRAM is needed to run all scenes. If you have less than that, hairball, powerplant & san-miguel scenes might crash the app.

Lucid demo is still in beta phase, so some bugs are expected, especially on AMD platforms, because most testing & development was done on NVIDIA GPU. In case of not enough memory, application may crash or work slowly. All rasterizer options except debug_* should work. Demo was tested on RTX 3050 TI, AMD RX 6700 XT, and AMD Radeon RX Vega 8.
